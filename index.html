Spotify Playlist Adder

Manage Spotify playlists effortlessly using Terraform!

About the Project

This project demonstrates how to automate playlist management on Spotify using Terraform. Add, remove, or customize playlists programmatically.

Features

  • Automate Spotify playlist creation.
  • Programmatically add songs to playlists.
  • Leverage Terraform for scalable configuration.
  • Easy integration with Spotify API.
